# Outputs Directory

This directory contains all generated outputs from your analysis.

## Structure

### figures/
Generated visualizations and plots
- Save all charts, graphs, and plots here
- Use descriptive names (e.g., `dissatisfaction_by_department.png`)
- Export in multiple formats if needed (PNG, SVG, PDF)

### reports/
Analysis reports and summaries
- Generated HTML/PDF reports
- Summary statistics
- Findings documentation

### models/
Saved models (if applicable)
- Trained machine learning models
- Model parameters and configurations
- Model performance metrics

## Best Practices
1. Use descriptive file names with dates
2. Include version numbers for iterative work
3. Document what each output represents
4. These files are typically excluded from version control
5. Can be regenerated from notebooks/scripts
